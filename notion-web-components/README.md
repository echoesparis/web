# Notion Components

Embed Notion content as interactive web components.

| Component | Description |
|-----------|-------------|
| **[Slideshow](https://maiastra.vercel.app/notion-slideshow/2bcf3868d02e8089ad26d58eb933e518)** | Coverflow gallery from Notion images |
| **[Typewriter](https://maiastra.vercel.app/notion-typewriter/2adf3868d02e808fb378cc73123f876c)** | Animated typewriter from Notion text |

### Usage

```
https://maiastra.vercel.app/notion-slideshow/{PAGE_ID}
https://maiastra.vercel.app/notion-typewriter/{PAGE_ID}
```

Share your Notion page with the integration, then use the page ID in the URL.
